初心所系

项目缘起

聚焦老人记忆留存难题，以科技之力构建记忆守护平台

目标人群

聚焦长者记忆珍藏，携手家属与社会机构共筑人生宝库

聚焦老人记忆留存难题，以科技之力构建记忆守护平台。在与社区银发长者的深度交流中，我们触摸到时光沉淀的重量——那些裹挟着方言乡音的故土记忆、家族变迁的跌宕故事、时代浪潮中的个体印记，正随着传统记录方式的式微逐渐模糊。「记忆小舟」的诞生，源于对「遗忘」的抵抗。我们深耕老年群体记忆传承困境，以AI技术为笔，为时光留白处填上温暖注脚，让每个生命故事都值得被郑重收藏。

  • 核心主角：60岁+长者（尤对方言使用者、有特殊人生经历者友好）
  • 同行伙伴：
    • 珍视家族根系的「记忆传承人」（子女/孙辈）
    • 深耕社区文化的「在地记录者」（街道办/养老机构）
    • 专注历史存档的「文明守望者」（档案馆/文化研究机构）
  • 特别场景：
    • 多代同堂家庭的「跨代记忆对话」
    • 地方文化机构的「方言口述史工程」
    • 养老社区的「长者生命故事集」

智忆传承

项目介绍

深耕老年记忆传承困境，借 AI 技术铸就智慧记忆宝库

核心价值

聚焦记忆传承使命，以 AI 赋能搭建全民叙事桥梁

「记忆小舟」是专为老年人设计的AI记忆采集系统，融合语音识别、情感计算、动态数据库等核心技术，以「对话式引导+智能化处理」模式，轻松实现记忆的数字化留存：

  • 适老化交互：数字人「记忆小舟」共情交互支持方言自然对话（如上海话、粤语、四川话），打破传统访谈的隔阂感
  • 一站式服务：一键完成「语音采集→文字转写→故事润色→档案生成」全流程，自动生成带时间轴的电子档案
  • 文化价值延伸：系统内置历史事件库（如抗战、改革开放等），自动关联讲述内容，让个人记忆升华为时代注脚
  • 对长者：
    • 打破「表达障碍」：AI助手化身耐心倾听者，让记忆讲述如日常聊天般轻松
    • 赋予「生命重量」：将碎片化回忆整合成可传承的「人生故事集」，收获「被看见」的尊严感
  • 对家庭：
    • 跨越代际的「情感纽带」：子女可通过系统回溯长辈人生，解锁「熟悉又陌生」的家族密码
    • 可触摸的「传家之宝」：电子档案+纸质纪念册双载体，让家风故事穿越时光
  • 对社会：
    • 构建「平民史观」：用AI抢救性记录濒危方言与个体记忆，为地方文化保护提供鲜活素材
    • 科技温度实践：探索「AI适老化」新范式，让银发群体成为技术红利的受益者而非旁观者
