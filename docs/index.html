项目缘起

聚焦老人记忆留存难题，以科技之力构建记忆守护平台

目标人群

聚焦长者记忆珍藏，携手家属与社会机构共筑人生宝库

“记忆小舟”源自我们对老年群体的长期田野调研。在与社区老人深度交流的过程中，我们发现，他们拥有宝贵的生命故事与人生智慧，但由于表达障碍、记录手段缺乏等原因，这些记忆常常无法被妥善保存。为回应这一真实需求，我们结合语音识别、自然语言处理与情感计算技术，打造出一款温柔可靠的AI语言助手。

我们的服务对象为60岁及以上、具备基本表达能力的长者，及其家属、社区机构、养老中心等希望进行记忆采集和整理的组织。

小舟载记，字字生光。岁岁留痕，代代相传。
